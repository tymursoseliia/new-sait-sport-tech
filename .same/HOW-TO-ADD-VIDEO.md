# 🎥 Как добавить видео на главную страницу

## ⚡ Быстрая инструкция (3 простых шага)

### Шаг 1: Загрузите видео

1. Откройте папку `public/videos/` в левой панели Same
2. Перетащите ваш видеофайл в эту папку
3. Переименуйте файл в **`hero-video.mp4`** (точно!)

**Требования к видео:**
- Формат: MP4
- Разрешение: 1920x1080 (рекомендуется)
- Размер: до 15 МБ
- Длительность: 10-30 секунд

---

### Шаг 2: Включите видео режим

Откройте файл `src/app/page.tsx` (строка ~36) и найдите:

```tsx
const USE_VIDEO = false; // ← Измените на true
```

Измените на:

```tsx
const USE_VIDEO = true; // ← Видео включено!
```

---

### Шаг 3: Сохраните и обновите страницу

1. Сохраните файл (Ctrl+S)
2. Обновите preview (F5 или Ctrl+R)
3. ✅ **Готово!** Видео проигрывается на главной

---

## 📖 Подробно о переключателе

В коде главной страницы есть простой переключатель:

```tsx
const USE_VIDEO = false; // false = изображение, true = видео
```

**Когда `false`:**
- ✅ Показывается красивое фоновое изображение
- ✅ Быстрая загрузка страницы
- ✅ Работает всегда

**Когда `true`:**
- 🎬 Показывается ваше видео
- 🔄 Автоматическое зацикливание
- 🔇 Без звука (muted)
- 📱 Работает на всех устройствах

## 🎯 Где взять видео

### Если у вас нет своего видео:

**Бесплатные стоки:**
- [Pexels Videos](https://www.pexels.com/videos/) - поиск: "luxury car"
- [Pixabay Videos](https://pixabay.com/videos/) - поиск: "car driving"

**Скачайте видео:**
1. Найдите подходящее видео
2. Скачайте в разрешении 1080p
3. Переименуйте в `hero-video.mp4`
4. Загрузите в `public/videos/`

---

## 🔧 Если видео не работает

### Проверьте:

1. **Имя файла точно `hero-video.mp4`** (не Hero-Video.mp4 или hero_video.mp4)
2. **Файл в папке `public/videos/`** (не в videos без public)
3. **Переключатель `USE_VIDEO = true`** в src/app/page.tsx
4. **Формат MP4** (H.264 кодек)
5. **Размер до 50 МБ** (лучше до 15 МБ)

### Консоль браузера:

Откройте консоль (F12) и проверьте:
- Если ошибка 404 → файл не найден, проверьте путь
- Если ошибка воспроизведения → пересохраните в MP4 H.264

---

## 💡 Совет: Сжатие видео

Если файл слишком большой, используйте:

**Онлайн сервисы (бесплатно):**
- [FreeConvert](https://www.freeconvert.com/video-compressor)
- [Clipchamp](https://clipchamp.com/en/video-compressor/)

**Целевые параметры:**
- Размер: 10-15 МБ
- Битрейт: 2-4 Mbps
- Разрешение: 1920x1080
- Кодек: H.264

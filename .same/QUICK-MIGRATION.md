## 🔑 ШАГ 1: Получите токен (2 минуты)

1. Откройте: **https://www.sanity.io/manage/personal/project/9pu75wlq/api/tokens**
2. Нажмите **"Add API Token"**
3. Name: `Migration Token`
4. Permissions: **Editor** ✅
5. Нажмите **"Create"** и скопируйте токен

## 💾 ШАГ 2: Добавьте токен

Откройте `.env.local` и добавьте строку:

```env
SANITY_WRITE_TOKEN=ваш_токен_сюда
```

## 🚀 ШАГ 3: Запустите миграцию

```bash
bun run migrate
```

**Готово!** Все 6 машин, 10 отзывов и 9 статей появятся в Sanity! ✅

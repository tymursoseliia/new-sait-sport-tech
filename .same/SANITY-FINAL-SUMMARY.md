## 🚀 Что нужно сделать ВРУЧНУЮ:

### Шаг 1: Создайте аккаунт на Sanity (5 минут)

1. Откройте https://www.sanity.io/
2. Нажмите **"Get started for free"**
3. Войдите через **GitHub** (рекомендуется)

---

### Шаг 2: Создайте проект (2 минуты)

1. После входа нажмите **"Create new project"**
2. Назовите проект: `ВОЛГА-АВТО`
3. Выберите **Dataset**: `production`
4. Нажмите **"Create project"**

---

### Шаг 3: Скопируйте Project ID (1 минута)

После создания проекта вы увидите **Project ID** (например: `abc123xyz`)

**Скопируйте этот ID!**

---

### Шаг 4: Добавьте Project ID в проект (2 минуты)

1. Откройте файл `.env.local` в корне проекта
2. Найдите строку:
   ```
   NEXT_PUBLIC_SANITY_PROJECT_ID=замените_на_ваш_project_id
   ```
3. Замените на ваш реальный Project ID:
   ```
   NEXT_PUBLIC_SANITY_PROJECT_ID=abc123xyz
   ```
4. Сохраните файл

---

### Шаг 5: Добавьте CORS Origins (2 минуты)

В Sanity Dashboard:

1. Откройте ваш проект
2. Перейдите в **API** → **CORS Origins**
3. Нажмите **"Add CORS origin"**
4. Добавьте:
   - `http://localhost:3000` (для разработки)
   - Ваш Vercel URL (для продакшена)
5. Включите **"Allow credentials"** ✅
6. Нажмите **"Save"**

---

### Шаг 6: Перезапустите сервер (1 минута)

```bash
# Остановите текущий сервер (Ctrl+C)
bun run dev
```

---

### Шаг 7: Откройте Sanity Studio (готово!)

Перейдите на **http://localhost:3000/studio**

🎉 **Вы увидите красивую админ-панель Sanity!**

---

## 📝 Как использовать Sanity Studio:

### Добавить автомобиль:

1. Откройте `/studio`
2. Войдите в свой аккаунт Sanity (если попросит)
3. Нажмите **"+"** → **"Автомобиль"**
4. Заполните поля:
   - Марка, модель, год
   - Пробег, цена
   - Загрузите фото (drag & drop)
   - Заполните slug (автозаполнение)
5. Нажмите **"Publish"**

### Проверить на сайте:

1. Откройте `/catalog`
2. Обновите страницу (F5)
3. ✅ Ваш автомобиль появился!

---

## 🔗 Куда попадают данные:

- **Автомобили** из Sanity → отображаются на `/catalog`
- **Отзывы** из Sanity → отображаются на `/reviews`
- **Статьи** из Sanity → отображаются на `/blog`

## 🌐 Развертывание на Vercel:

### После настройки Sanity локально:

1. Откройте Vercel Dashboard
2. Перейдите в **Settings** → **Environment Variables**
3. Добавьте:
   ```
   NEXT_PUBLIC_SANITY_PROJECT_ID = ваш_project_id_здесь
   NEXT_PUBLIC_SANITY_DATASET = production
   ```
4. Нажмите **"Save"**
5. **Redeploy** проект

✅ Теперь админ-панель работает и на продакшене!

## ❓ FAQ:

**Q: Что если я не настрою Sanity?**
A: Сайт продолжит работать со статическими данными. Sanity опционален.

**Q: Можно ли использовать бесплатно?**
A: Да! Бесплатный план Sanity поддерживает до 3 пользователей и 100k запросов/месяц.

**Q: Нужно ли удалять старую админ-панель (/admin)?**
A: Можно удалить, но не обязательно. Sanity Studio (/studio) - это замена.

**Q: Как добавить других администраторов?**
A: В Sanity Dashboard → **Settings** → **Members** → **Invite**

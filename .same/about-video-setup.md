# Настройка фонового видео на странице "О компании"

**ВАЖНО:** Видео добавлено в ВЕРХНЮЮ секцию страницы (Hero), а НЕ в секцию с историей компании.

- ✅ Overlay режим: **light** (40-20% затемнения)
- ✅ Видео воспроизводится автоматически на странице `/about`

## Как загрузить видео

### 1. Подготовьте видеофайл

**Рекомендации:**
- Формат: MP4 (H.264)
- Разрешение: Full HD (1920x1080)
- Длительность: 10-30 секунд (будет проигрываться в цикле)
- Размер: до 10 МБ
- Содержание: команда за работой, офис, процесс подбора автомобилей

### 2. Загрузите файл

Поместите видео в: `public/videos/about-bg.mp4`

### 3. Готово!

Видео автоматически начнет воспроизводиться на странице `/about` в Hero секции.

**Переключатель уже включен:**
```tsx
const USE_VIDEO = true; // Видео включено по умолчанию
```

## Если нужно выключить видео

Откройте файл `src/app/about/page.tsx` (примерно строка 165) и измените:

```tsx
const USE_VIDEO = false; // Видео выключено - показывается изображение
```

## Технические детали

### Где находится видеофон?

**Файл:** `src/app/about/page.tsx`
**Секция:** Hero (строки 162-188)
**Заголовок:** "О компании ООО 'Волга-Авто'"

### Атрибуты видео
- `autoPlay` - автоматический запуск
- `loop` - циклическое проигрывание
- `muted` - без звука (обязательно для autoplay)
- `playsInline` - проигрывание inline на iOS

### Стили
- Видео: `opacity-70` для прозрачности (70% - хорошо видно)
- Overlay: градиент `from-zinc-900/40 to-zinc-900/20` (light режим)
- Текст: светлый (`text-white`, `text-zinc-300`)
- Адаптивность: `overflow-hidden`, `relative`, `z-10`

### Fallback
Если видео не загрузится, автоматически показывается изображение с Unsplash.

## Оптимизация видео

### FFmpeg команда для сжатия:

```bash
ffmpeg -i input.mp4 -c:v libx264 -crf 28 -preset slow -vf scale=1920:-2 -an output.mp4
```

**Параметры:**
- `-crf 28` - качество (18-28)
- `-preset slow` - режим сжатия
- `-vf scale=1920:-2` - Full HD
- `-an` - удаление аудио

### Онлайн инструменты:
- https://www.freeconvert.com/video-compressor
- https://www.videosmaller.com/

## Известные проблемы

1. **Видео не проигрывается** - проверьте формат (должен быть MP4)
2. **Autoplay заблокирован** - атрибут `muted` обязателен
3. **iOS Safari** - требуется `playsInline`

## Часто задаваемые вопросы

**Q: Почему показывается изображение, а не видео?**
A: Видеофайл `public/videos/about-bg.mp4` еще не загружен. После загрузки видео появится автоматически.

**Q: Можно ли использовать другое видео?**
A: Да, измените путь в `videoSrc="/videos/your-video.mp4"`

**Q: Как изменить степень затемнения?**
A: Измените `overlayOpacity` на `'light'`, `'medium'` или `'dark'`

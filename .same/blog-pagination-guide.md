## 🔧 Как использовать

### Навигация между страницами

1. **Кнопка "Следующая"** - переход на следующую страницу
2. **Кнопка "Предыдущая"** - возврат на предыдущую страницу
3. **Номера страниц** - прямой переход на нужную страницу
4. **URL параметр** - `/blog?page=2` для прямого доступа

### Примеры URL
```
/blog           → Страница 1 (статьи 1-3)
/blog?page=1    → Страница 1 (статьи 1-3)
/blog?page=2    → Страница 2 (статьи 4-6)
/blog?page=3    → Страница 3 (статьи 7-9)
```

## 💻 Техническая реализация

### Код (упрощенно)
```tsx
const ARTICLES_PER_PAGE = 3;
const currentPage = Number(searchParams.page) || 1;
const totalPages = Math.ceil(articles.length / ARTICLES_PER_PAGE);

const startIndex = (currentPage - 1) * ARTICLES_PER_PAGE;
const endIndex = startIndex + ARTICLES_PER_PAGE;
const currentArticles = articles.slice(startIndex, endIndex);
```

### Логика пагинации
- **startIndex**: `(currentPage - 1) * 3`
  - Страница 1: `(1-1) * 3 = 0`
  - Страница 2: `(2-1) * 3 = 3`
  - Страница 3: `(3-1) * 3 = 6`

- **endIndex**: `startIndex + 3`
  - Страница 1: `0 + 3 = 3` (статьи 0-2)
  - Страница 2: `3 + 3 = 6` (статьи 3-5)
  - Страница 3: `6 + 3 = 9` (статьи 6-8)

## 🛠️ Как изменить количество статей на странице

Откройте файл `src/app/blog/page.tsx` и измените константу:

```tsx
// Было: 3 статьи на странице
const ARTICLES_PER_PAGE = 3;

// Можно изменить на:
const ARTICLES_PER_PAGE = 6; // 6 статей = 2 страницы
const ARTICLES_PER_PAGE = 4; // 4 статьи = 3 страницы
```

## 📝 Как добавить новые статьи

1. Откройте `src/app/blog/page.tsx`
2. Найдите массив `articles`
3. Добавьте новый объект:

```tsx
{
  id: 10, // следующий ID
  title: 'Ваш заголовок',
  excerpt: 'Краткое описание',
  image: 'https://example.com/image.jpg',
  date: '2024-11-17',
  category: 'Руководства',
  readTime: '15 мин',
}
```

Пагинация автоматически пересчитается!

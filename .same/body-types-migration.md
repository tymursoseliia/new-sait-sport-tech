## Как запустить миграцию

### Шаг 1: Получить SANITY_API_TOKEN

1. Откройте https://sanity.io/manage
2. Выберите ваш проект
3. Перейдите в API → Tokens
4. Создайте новый токен с правами **Write**
5. Скопируйте токен

### Шаг 2: Добавить токен в .env.local

```bash
SANITY_API_TOKEN=your_write_token_here
```

### Шаг 3: Запустить скрипт миграции

```bash
cd Волга-Авто-test
bun run scripts/migrate-body-types.ts
```

## Что делает скрипт

1. Получает все машины из Sanity
2. Находит машины с русскими значениями body
3. Обновляет их на английские:
   - Седан → sedan
   - Внедорожник → suv
   - Универсал → wagon
   - Купе → coupe
   - Минивэн → minivan
   - Хэтчбек → hatchback
   - Кроссовер → crossover
4. Выводит отчет о проделанной работе

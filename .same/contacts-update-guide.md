## 📝 Как изменить контакты в будущем

1. Откройте файл `src/config/contacts.ts`
2. Найдите нужное поле (phone, email, whatsapp, telegram)
3. Измените значение
4. Сохраните файл
5. **Готово!** Изменения применятся на всем сайте

### Пример:
```typescript

  phone: '+7 (901) 078-14-52',  // ← Измените здесь
```

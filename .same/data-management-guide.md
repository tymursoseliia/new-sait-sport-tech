# Руководство по управлению данными сайта

## 📋 Обзор

Все данные сайта теперь централизованы в отдельных файлах, что позволяет легко управлять контентом без редактирования компонентов.

---

## 🚗 Добавление/редактирование автомобилей

### Файл: `src/data/cars.ts`

### Как добавить новое авто:

1. Откройте файл `src/data/cars.ts`
2. Добавьте новый объект в массив `CARS`:

```typescript
{
  id: 7,                                    // Уникальный ID (следующий по порядку)
  make: 'Mercedes-Benz',                    // Марка
  model: 'E-Class',                         // Модель
  year: 2021,                               // Год
  mileage: 45000,                           // Пробег в км
  fuel: 'Дизель',                           // Тип топлива
  transmission: 'Автомат',                  // Коробка передач
  body: 'Седан',                            // Тип кузова
  price: 2950000,                           // Цена в рублях
  imageUrl: 'https://...',                  // URL изображения
  status: 'available',                      // 'available' или 'order'
  slug: 'mercedes-e-class-2021',           // Slug для URL (опционально)
  originCountry: 'Германия',               // Страна покупки (опционально)
},
```

### Поля автомобиля:

#### Обязательные поля:
- `id` (number) - уникальный идентификатор
- `make` (string) - марка автомобиля
- `model` (string) - модель
- `year` (number) - год выпуска
- `mileage` (number) - пробег в км
- `fuel` (string) - тип топлива
- `transmission` (string) - коробка передач
- `body` (string) - тип кузова (Седан, Внедорожник, Универсал, Купе)
- `price` (number) - цена в рублях
- `imageUrl` (string) - URL изображения
- `status` (CarStatus) - статус: `'available'` (в наличии) или `'order'` (под заказ)

#### Опциональные поля:
- `title` (string) - маркетинговый заголовок
- `drive` (string) - привод (4х4, Передний, Задний)
- `originCountry` (string) - страна покупки
- `city` (string) - город клиента/доставки
- `slug` (string) - slug для детальной страницы
- `tags` (string[]) - дополнительные теги
- `description` (string) - описание

### Где отображаются автомобили:
- Главная страница: первые 3 авто из массива CARS
- Страница каталога: все автомобили с фильтрацией

## ⭐ Добавление/редактирование отзывов

### Файл: `src/data/reviews.ts`

### Как добавить новый отзыв:

1. Откройте файл `src/data/reviews.ts`
2. Добавьте новый объект в массив `REVIEWS`:

```typescript
{
  id: 11,                                   // Уникальный ID (следующий по порядку)
  name: 'Иван П.',                          // Имя клиента
  city: 'Казань',                           // Город
  car: 'BMW X5',                            // Название автомобиля
  brand: 'BMW',                             // Марка (для фильтрации)
  date: '2024-11-20',                       // Дата в формате YYYY-MM-DD
  rating: 5,                                // Оценка от 1 до 5
  text: 'Отличный сервис...',               // Полный текст отзыва
  imageUrl: 'https://...',                  // URL фото (опционально)
  country: 'Германия',                      // Страна покупки (опционально)
  verified: true,                           // Верифицированный отзыв
  purchaseVerified: true,                   // Подтвержденная покупка
  highlights: {                             // Важные моменты (опционально)
    savings: '~600 000 ₽',                 // Экономия
    time: '10 дней',                       // Срок
    repeat: true,                           // Повторное обращение
  },
},
```

### Поля отзыва:

#### Обязательные поля:
- `id` (number) - уникальный идентификатор
- `name` (string) - имя клиента
- `city` (string) - город клиента
- `car` (string) - название автомобиля
- `brand` (string) - марка для фильтрации
- `date` (string) - дата в формате YYYY-MM-DD
- `rating` (number) - оценка от 1 до 5
- `text` (string) - текст отзыва

#### Опциональные поля:
- `imageUrl` (string) - URL изображения
- `country` (string) - страна покупки
- `verified` (boolean) - верифицированный отзыв
- `purchaseVerified` (boolean) - подтвержденная покупка
- `route` (string) - маршрут доставки
- `isGiveaway` (boolean) - участник розыгрыша
- `highlights` (object) - важные моменты:
  - `savings` (string) - экономия
  - `time` (string) - срок выполнения
  - `repeat` (boolean) - повторное обращение

### Где отображаются отзывы:
- Страница отзывов: все отзывы с фильтрацией по марке и городу

## 📞 Изменение контактов

### Файл: `src/config/contacts.ts`

### Как изменить контактные данные:

1. Откройте файл `src/config/contacts.ts`
2. Отредактируйте нужные поля в объекте `CONTACTS`

  // Адрес

  address: 'г. Тольятти, ул. Маршала Жукова, д. 54, этаж 1 офис 1 ком. 3',

### Где используются контакты:

**Все изменения применяются автоматически на всём сайте!**

## 🔄 Процесс обновления данных

### Для добавления нового авто:

1. Откройте `src/data/cars.ts`
2. Добавьте объект в массив `CARS`
3. Сохраните файл
4. Задеплойте изменения

### Для добавления нового отзыва:

1. Откройте `src/data/reviews.ts`
2. Добавьте объект в массив `REVIEWS`
3. Сохраните файл
4. Задеплойте изменения

### Для изменения контактов:

1. Откройте `src/config/contacts.ts`
2. Измените нужные поля в объекте `CONTACTS`
3. Сохраните файл
4. Задеплойте изменения

## ✅ Преимущества новой структуры

1. **Простота**: все данные в одном месте
2. **Безопасность**: не нужно редактировать компоненты React
3. **Типизация**: TypeScript проверяет правильность данных
4. **Консистентность**: контакты одинаковые везде
5. **Скорость**: быстрое добавление нового контента

---

## 📝 Примечания

- Всегда проверяйте типизацию при добавлении новых данных
- ID автомобилей и отзывов должны быть уникальными
- Формат даты для отзывов: `YYYY-MM-DD` (например, `2024-11-20`)
- URL изображений должны быть доступны публично
- При изменении контактов проверьте JSON-LD микроразметку

## 🎨 Как это работает

### Компонент Logo.tsx

Логотип автоматически переключается между светлым и темным вариантом при изменении темы:

```tsx
// Светлый логотип - показывается на светлой теме
<img
  src="https://ugc.same-assets.com/D69LZBrW5zr4b1E1nzzHZUrfkQHP7KoP.png"
  alt="ВОЛГА-АВТО"
  className="h-16 md:h-20 w-auto transition-all duration-300 dark:hidden"
/>

// Темный логотип - показывается на темной теме
<img
  src="/logo-dark.png"
  alt="ВОЛГА-АВТО"
  className="h-16 md:h-20 w-auto transition-all duration-300 hidden dark:block"
/>
```

### Tailwind CSS классы

- `dark:hidden` - скрывает элемент в темной теме
- `hidden dark:block` - скрывает элемент в светлой теме, показывает в темной
- `transition-all duration-300` - плавная анимация переключения (300ms)

## 🔄 Процесс переключения

1. **Пользователь кликает** на переключатель темы (иконка солнца/луны в Header)
2. **ThemeContext обновляет** состояние темы (`light` → `dark` или наоборот)
3. **Tailwind CSS применяет** класс `dark` к корневому элементу
4. **Logo компонент автоматически** переключает логотипы через CSS
5. **Анимация** длится 300ms для плавного перехода

## 🎯 Технические детали

### Предотвращение Hydration Mismatch

Компонент использует `mounted` state для предотвращения ошибок гидратации:

```tsx
const [mounted, setMounted] = useState(false);

useEffect(() => {
  setMounted(true);
}, []);

// До монтажа - показываем только светлый логотип
if (!mounted) {
  return <img src="светлый_логотип" />;
}

// После монтажа - показываем оба логотипа с переключением
return (
  <>
    <img className="dark:hidden" src="светлый_логотип" />
    <img className="hidden dark:block" src="темный_логотип" />
  </>
);
```

### Почему это важно?

- **SSR (Server-Side Rendering)**: При первом рендере на сервере тема неизвестна
- **Клиент**: После монтажа компонент получает актуальную тему из ThemeContext
- **Без мерцания**: Плавное переключение без визуальных скачков

## 🚀 Как протестировать

1. Откройте сайт в браузере
2. Найдите переключатель темы в правом верхнем углу Header
3. Кликните на иконку солнца/луны
4. **Логотип должен плавно переключиться** с одного варианта на другой

**Ожидаемый результат:**
- Светлая тема → видим обычный логотип
- Темная тема → видим темный логотип с белым/светлым текстом

## 🔧 Как изменить логотипы

### Заменить темный логотип:

1. Подготовьте новое изображение (формат PNG, рекомендуемый размер: 1000x500px)
2. Сохраните файл как `/public/logo-dark.png`
3. Обновите страницу - новый логотип применится автоматически

### Заменить светлый логотип:

1. Загрузите новое изображение на хостинг (например, same-assets.com)
2. Откройте `src/components/Logo.tsx`
3. Замените URL в обоих местах (строки 18 и 31):
   ```tsx
   src="https://новый_url_логотипа.png"
   ```

## 🚀 ЧТО НУЖНО СДЕЛАТЬ СЕЙЧАС:

### Шаг 1: Настройте CORS Origins в Sanity Dashboard (ОБЯЗАТЕЛЬНО!)

Это **критически важно** для работы Sanity Studio!

1. Откройте https://www.sanity.io/manage
2. Войдите в свой аккаунт (через GitHub)
3. Выберите ваш проект **ВОЛГА-АВТО** (Project ID: `9pu75wlq`)
4. Перейдите в **API** → **CORS Origins**
5. Нажмите **"Add CORS origin"**
6. Добавьте:
   ```
   http://localhost:3000
   ```
7. Включите галочку **"Allow credentials"** ✅
8. Нажмите **"Save"**

**Без этого шага Sanity Studio НЕ БУДЕТ работать!**

---

### Шаг 2: Откройте Sanity Studio

После настройки CORS:

1. Откройте в браузере: **http://localhost:3000/studio**
2. Вы увидите страницу входа Sanity
3. Войдите через ваш GitHub аккаунт (или тот, что использовали для регистрации)

### Шаг 3: Добавьте первые данные

В Sanity Studio:

#### Добавить автомобиль:
1. Нажмите кнопку **"+"** (Create new document)
2. Выберите **"Автомобиль"**
3. Заполните поля:
   - Марка: `BMW`
   - Модель: `X5`
   - Год: `2022`
   - Пробег: `45000`
   - Топливо: `Бензин`
   - КПП: `Автомат`
   - Кузов: `Внедорожник`
   - Цена: `4500000`
   - Статус: `В наличии`
4. Загрузите фото (drag & drop)
5. Нажмите **"Generate"** рядом со slug (автоматическая генерация)
6. Нажмите **"Publish"**

#### Проверить на сайте:
1. Откройте **http://localhost:3000/catalog**
2. Обновите страницу (F5)
3. ✅ Ваш BMW X5 должен появиться в каталоге!

## 📝 Как работает Sanity:

### Приоритет данных:
1. **Если в Sanity есть данные** → используются данные из Sanity
2. **Если в Sanity пусто** → используются статические данные (fallback)

Это значит, что сайт будет работать даже если Sanity не настроен!

## 🌐 Развертывание на Vercel:

Когда будете готовы развернуть сайт на продакшене:

### 1. Добавьте переменные окружения в Vercel:

1. Откройте Vercel Dashboard → ваш проект
2. Перейдите в **Settings** → **Environment Variables**
3. Добавьте:
   ```
   NEXT_PUBLIC_SANITY_PROJECT_ID = 9pu75wlq
   NEXT_PUBLIC_SANITY_DATASET = production
   ```
4. Нажмите **"Save"**

### 2. Добавьте CORS Origin для Vercel:

В Sanity Dashboard:
1. **API** → **CORS Origins** → **"Add CORS origin"**
2. Добавьте ваш Vercel URL (например: `https://volga-auto-xxx.vercel.app`)
3. Включите **"Allow credentials"** ✅
4. Нажмите **"Save"**

### 3. Redeploy проекта:

В Vercel Dashboard:
1. Перейдите в **Deployments**
2. Нажмите на последний деплой → **"Redeploy"**
3. ✅ Готово! Sanity Studio работает на продакшене

## ❓ Частые вопросы:

**Q: Что делать, если /studio показывает ошибку?**
A: Убедитесь, что добавили CORS Origin в Sanity Dashboard (Шаг 1)

**Q: Нужно ли удалять старую админ-панель (/admin)?**
A: Можно оставить для совместимости, но рекомендуется использовать Sanity Studio (/studio)

**Q: Можно ли импортировать данные из старой админ-панели?**
A: Да! Вручную скопируйте данные из JSON файлов (`src/data/db/`) в Sanity Studio

**Q: Как добавить других администраторов?**
A: Sanity Dashboard → **Settings** → **Members** → **Invite**

**Q: Сколько стоит Sanity?**
A: Бесплатно до 3 пользователей и 100k запросов/месяц

## 📊 Что реализовано

### Форма отзывов (`ReviewForm.tsx`)

Полнофункциональная форма для сбора отзывов от клиентов с интеграцией Telegram.

**Поля формы:**
- ✅ **Имя** (обязательное)
- ✅ **Город** (обязательное)
- ✅ **Автомобиль** (обязательное)
- ✅ **Email** (опциональное) - для обратной связи
- ✅ **Фото автомобиля** (опциональное) - ссылка на фото
- ✅ **Рейтинг** (обязательное) - 1-5 звезд с интерактивным выбором
- ✅ **Текст отзыва** (обязательное) - подробный отзыв
- ✅ **Согласие на обработку данных** (обязательное)

**Возможности:**
- 🔄 Валидация всех обязательных полей
- 📱 Отправка уведомления в Telegram
- ✨ Интерактивный выбор рейтинга (hover эффект)
- ⏱️ Автоматический сброс формы через 3 секунды после успешной отправки
- 🎨 Визуальные статусы: загрузка, успех, ошибка
- 🔒 Защита от повторной отправки во время обработки

## 🚀 Где находится форма

### На странице отзывов
**URL:** `/reviews#review-form`

Форма расположена внизу страницы отзывов в отдельной секции.

**Навигация к форме:**
1. Перейти на страницу "Отзывы" через меню

## 📨 Telegram интеграция

### Формат сообщения

При отправке отзыва в Telegram приходит сообщение:

```
📝 НОВЫЙ ОТЗЫВ

👤 Имя: [имя клиента]
🏙️ Город: [город]
🚗 Автомобиль: [марка модель год]
⭐ Оценка: ★★★★★ (5/5)
📧 Email: [email] (если указан)

💬 Текст отзыва:
[текст отзыва клиента]

📸 Фото автомобиля: [ссылка] (если указано)

—————————————————
Дата: [дата и время]
```

### Настройка Telegram

Конфигурация находится в `src/config/telegram.ts`:
- Bot Token: из переменной окружения `TELEGRAM_BOT_TOKEN`
- Chat ID: из переменной окружения `TELEGRAM_CHAT_ID`

**Файл `.env.local`** (создайте если нет):
```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
TELEGRAM_CHAT_ID=ваш_chat_id
```

## 🧪 Как протестировать

### Шаг 1: Открыть форму
1. Откройте preview сайта
2. Перейдите в раздел **Отзывы**
3. Прокрутите вниз до формы

### Шаг 2: Заполнить тестовые данные

**Вариант 1: Полная форма (все поля)**
```
Имя: Тестовый Клиент
Город: Москва
Автомобиль: BMW X5 2021
Email: test@example.com
Фото: https://example.com/photo.jpg
Рейтинг: ⭐⭐⭐⭐⭐ (5 звезд)
Текст: Отличная работа! Помогли подобрать идеальный автомобиль.
```

**Вариант 2: Минимальная форма (только обязательные)**
```
Имя: Краткий Тест
Город: СПб
Автомобиль: Toyota Camry
Рейтинг: ⭐⭐⭐⭐ (4 звезды)
Текст: Всё прошло хорошо, спасибо!
```

### Шаг 3: Отправить и проверить Telegram
1. Поставьте галочку на согласие с обработкой данных
2. Нажмите "Отправить отзыв"
3. Дождитесь зеленого сообщения об успехе
4. Проверьте Telegram - должно прийти уведомление

## 🔄 Workflow: От отправки до публикации

### 1. Клиент заполняет форму
- Клиент оставляет отзыв на сайте через форму

### 2. Уведомление в Telegram
- Отзыв приходит в ваш Telegram
- Вы проверяете отзыв на адекватность

### 3. Модерация
- Если отзыв подходит - переходите к публикации
- Если нет - игнорируете

### 4. Публикация отзыва
Откройте `src/data/reviews.ts` и добавьте новый отзыв:

```typescript
{
  id: 12, // следующий ID
  name: 'Имя из Telegram',
  city: 'Город из Telegram',
  car: 'Автомобиль из Telegram',
  brand: 'BMW', // извлеките марку
  country: 'Германия', // укажите страну
  rating: 5, // из Telegram
  text: 'Текст отзыва из Telegram',
  date: '2025-11-16', // текущая дата
  verified: true,
  purchaseVerified: true,
  imageUrl: 'ссылка на фото', // если было в отзыве
  highlights: {
    savings: '500 000 ₽', // опционально
    time: '14 дней', // опционально
  },
},
```

### 5. Деплой
После добавления отзыва в файл:
```bash
git add .
git commit -m "Добавлен новый отзыв"
git push
```

Или используйте админ-панель `/admin` для копирования шаблона.

## ⚠️ Известные особенности

### Hydration Warning
В консоли может появляться предупреждение о hydration mismatch. Это известная особенность Next.js и не влияет на функциональность - React автоматически исправляет это на клиенте.

### Загрузка изображений
Некоторые изображения могут не загружаться (ugc.same-assets.com) - это временные тестовые ссылки. В продакшене замените их на реальные URL.

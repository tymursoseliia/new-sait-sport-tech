## 📝 Что нужно сделать:

### Шаг 1: Создайте аккаунт на Sanity

1. Откройте https://www.sanity.io/
2. Нажмите **"Sign up"** или **"Get started"**
3. Войдите через GitHub (рекомендуется) или email

---

### Шаг 2: Создайте новый проект

1. После входа нажмите **"Create new project"**
2. Заполните:
   - **Project name:** `ВОЛГА-АВТО`
   - **Dataset:** `production` (оставьте по умолчанию)
3. Нажмите **"Create project"**

---

### Шаг 3: Получите Project ID

1. После создания проекта вы увидите **Project ID**
2. Скопируйте этот ID (например: `abc123xyz`)

---

### Шаг 4: Создайте токен доступа (опционально, для редактирования)

1. Перейдите в **API** → **Tokens**
2. Нажмите **"Add API token"**
3. Заполните:
   - **Label:** `Next.js App`
   - **Permissions:** `Editor` или `Write`
4. Нажмите **"Save"**
5. Скопируйте токен (показывается только один раз!)

---

### Шаг 5: Добавьте переменные в .env.local

Откройте файл `.env.local` и добавьте:

```env
# Sanity CMS
NEXT_PUBLIC_SANITY_PROJECT_ID=ВАШ_PROJECT_ID_ЗДЕСЬ
NEXT_PUBLIC_SANITY_DATASET=production
SANITY_API_TOKEN=ВАШ_ТОКЕН_ЗДЕСЬ (если создали)
```

**Замените:**
- `ВАШ_PROJECT_ID_ЗДЕСЬ` → скопированный Project ID
- `ВАШ_ТОКЕН_ЗДЕСЬ` → скопированный токен (если нужно)

---

### Шаг 6: Добавьте домен в CORS Origins

1. В Sanity Dashboard перейдите в **API** → **CORS Origins**
2. Нажмите **"Add CORS origin"**
3. Добавьте:
   - `http://localhost:3000` (для разработки)
   - Ваш Vercel домен (для продакшена, например: `https://volga-auto-xxxx.vercel.app`)
4. Разрешите **credentials**: ✅

---

### Шаг 7: Запустите Sanity Studio

После добавления переменных окружения:

```bash
cd Волга-Авто
bun run dev
```

Откройте: **http://localhost:3000/studio**

Вы увидите Sanity Studio - это и есть ваша админ-панель! 🎉

---

## 🎯 Как использовать Sanity Studio:

### Добавить автомобиль:

1. Откройте `/studio`
2. Нажмите **"Create"** → **"Автомобиль"**
3. Заполните все поля
4. Загрузите фото (drag-and-drop)
5. Нажмите **"Publish"**

### Добавить отзыв:

1. Нажмите **"Create"** → **"Отзыв"**
2. Заполните данные клиента
3. Поставьте рейтинг (1-5 звёзд)
4. Нажмите **"Publish"**

### Добавить статью:

1. Нажмите **"Create"** → **"Статья"**
2. Заполните заголовок
3. Добавьте содержание (rich text editor)
4. Загрузите изображение
5. Нажмите **"Publish"**

## 🔗 Интеграция с сайтом:

- **Статьи** → `/blog`

После публикации в Sanity Studio - обновите страницу сайта!

## 🚀 Развертывание на Vercel:

### В Vercel добавьте переменные окружения:

1. Откройте **Settings** → **Environment Variables**
2. Добавьте:
   ```
   NEXT_PUBLIC_SANITY_PROJECT_ID = ваш_project_id
   NEXT_PUBLIC_SANITY_DATASET = production
   SANITY_API_TOKEN = ваш_токен (если нужно)
   ```
3. Нажмите **Save**
4. **Redeploy** проект

4. **Кэширование статики:**
   ```javascript
   headers() {
     return [
       // Изображения: кэш 1 год, immutable

       // _next/static: кэш 1 год, immutable

     ];
   }
   ```

5. **Форматы изображений:**
   - WebP (поддержка 95%+ браузеров)
   - AVIF (еще более сжатый формат)

## 🚀 Пошаговая настройка (ОБЯЗАТЕЛЬНО!)

### Шаг 1: Создание Telegram бота

1. **Откройте Telegram** и найдите [@BotFather](https://t.me/botfather)

2. **Создайте бота:**
   - Отправьте команду: `/newbot`
   - Введите название бота: `ВОЛГА-АВТО Заявки` (или любое другое)
   - Введите username бота: `volga-auto_premier_bot` (должен заканчиваться на `_bot`)

3. **Сохраните токен бота:**
   - BotFather пришлёт сообщение с токеном
   - Токен выглядит так: `6234567890:ABCdefGHIjklMNOpqrsTUVwxyz1234567890`
   - **ВАЖНО:** Никому не показывайте этот токен!

### Шаг 2: Получение Chat ID

1. **Найдите бота** [@userinfobot](https://t.me/userinfobot)

2. **Отправьте ему любое сообщение**

3. **Скопируйте ваш Chat ID**
   - Он выглядит как число: `123456789` или `-123456789`
   - Это ваш личный ID в Telegram

### Шаг 3: Настройка переменных окружения

**Вариант А: Через файл .env.local (Рекомендуется для production)**

1. Создайте файл `.env.local` в корне проекта:

```bash
# Скопируйте пример файла
cp .env.local.example .env.local
```

2. Откройте `.env.local` и замените значения:

```env
TELEGRAM_BOT_TOKEN=6234567890:ABCdefGHIjklMNOpqrsTUVwxyz1234567890
TELEGRAM_CHAT_ID=123456789
```

3. Перезапустите сервер разработки

**Вариант Б: Напрямую в коде (Только для тестирования!)**

1. Откройте файл `src/config/telegram.ts`

2. Замените строки:

```typescript
botToken: process.env.TELEGRAM_BOT_TOKEN || 'ВАШ_ТОКЕН_СЮДА',
chatId: process.env.TELEGRAM_CHAT_ID || 'ВАШ_CHAT_ID_СЮДА',
```

⚠️ **ВНИМАНИЕ:** Не коммитьте файл с реальными токенами в Git!

### Шаг 4: Активация бота

1. **Откройте вашего бота** в Telegram (по username, который вы создали)

2. **Нажмите START** или отправьте команду `/start`

3. Теперь бот готов принимать сообщения!

### Шаг 5: Тестирование

1. Откройте сайт в браузере

2. Перейдите на любую страницу с формой:
   - Главная страница
   - Страница контактов
   - Каталог (кнопка "Оставить заявку")

3. Заполните форму:
   - Имя: Тест
   - Телефон: +7 999 123 45 67
   - Сообщение: Проверка Telegram интеграции

4. Нажмите "Отправить заявку"

5. **Проверьте Telegram** - вы должны получить сообщение вида:


## ⚙️ Настройки и кастомизация

### Изменить формат сообщения

Откройте `src/config/telegram.ts` и отредактируйте функцию `formatTelegramMessage`:

```typescript
export const formatTelegramMessage = (data) => {
  let message = `🚗 <b>Новая заявка</b>\n\n`;
  // ... добавьте свои поля
  return message;
};
```

### Отправка в несколько чатов

Если нужно отправлять в несколько Telegram чатов/каналов:

1. Откройте `src/config/telegram.ts`

2. Измените `chatId` на массив:

```typescript
chatId: process.env.TELEGRAM_CHAT_ID || '123456789,987654321',
```

3. Обновите API route `src/app/api/send-to-telegram/route.ts`:

```typescript
const chatIds = TELEGRAM_CONFIG.chatId.split(',');

for (const chatId of chatIds) {
  await fetch(telegramUrl, {
    // ...
    body: JSON.stringify({
      chat_id: chatId.trim(),
      // ...
    }),
  });
}
```

## 🐛 Troubleshooting (Решение проблем)

### Проблема: "Telegram не настроен"

**Решение:**
- Проверьте, что `.env.local` создан и содержит правильные значения
- Перезапустите сервер: `bun run dev`
- Убедитесь, что токен и Chat ID скопированы правильно

### Проблема: Сообщения не приходят

**Решение:**
1. Проверьте, что вы нажали START в боте
2. Убедитесь, что Chat ID верный (отправьте сообщение @userinfobot)
3. Проверьте консоль браузера на ошибки (F12 → Console)
4. Проверьте логи сервера

### Проблема: Ошибка 401 Unauthorized

**Решение:**
- Токен бота неверный
- Создайте нового бота у @BotFather
- Скопируйте новый токен

### Проблема: Ошибка 400 Bad Request

**Решение:**
- Chat ID неверный или в неправильном формате
- Проверьте Chat ID у @userinfobot
- Убедитесь, что нет лишних пробелов

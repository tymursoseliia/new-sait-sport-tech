## 🔍 Технические детали анимации

### Staggered Delays (Каскадные задержки)

```css
html.theme-transitioning section:nth-of-type(1) {
  transition-delay: 0ms !important;
}

html.theme-transitioning section:nth-of-type(2) {
  transition-delay: 100ms !important;
}

html.theme-transitioning section:nth-of-type(3) {
  transition-delay: 200ms !important;
}

/* ... до 7 секций */
```

**Результат:** ✅ Волна движется сверху вниз с интервалом 100ms

### Transition Properties

```css
section, header, footer, main > div, .card {
  transition: background-color 0.6s cubic-bezier(0.4, 0, 0.2, 1),
              border-color 0.6s cubic-bezier(0.4, 0, 0.2, 1),
              color 0.6s cubic-bezier(0.4, 0, 0.2, 1);
}
```

**Результат:** ✅ Плавные переходы с easing функцией

### Theme Switching Logic

```tsx
const updateTheme = (newTheme: Theme) => {
  const root = document.documentElement;

  root.classList.add('theme-transitioning');

  if (newTheme === 'dark') {
    root.classList.add('dark');
  } else {
    root.classList.remove('dark');
  }

  setTimeout(() => {
    root.classList.remove('theme-transitioning');
  }, 1000);
};
```

**Результат:** ✅ Класс `theme-transitioning` активирует анимацию на 1 секунду

## 🔧 Как работает

### 1. Первая загрузка
При первом посещении сайта:
1. Проверяется `localStorage` на наличие сохраненной темы
2. Если темы нет - проверяются системные настройки браузера (`prefers-color-scheme: dark`)
3. Применяется тема БЕЗ мерцания (благодаря скрипту предзагрузки)

### 2. Переключение темы
При клике на кнопку:
1. Тема переключается между `light` и `dark`
2. Новая тема сохраняется в `localStorage`
3. Применяется класс `.dark` к `<html>` элементу
4. Все CSS переменные обновляются автоматически

### 3. Сохранение
Выбранная тема сохраняется и будет применяться при всех последующих посещениях.

## 🎨 CSS Переменные

### Светлая тема (`:root`)
```css
--background: 0 0% 100%;           /* Белый фон */
--foreground: 0 0% 9%;             /* Почти черный текст */
--card: 0 0% 100%;                 /* Белые карточки */
--border: 0 0% 90%;                /* Светлые границы */
--muted: 0 0% 96%;                 /* Приглушенный фон */
--muted-foreground: 0 0% 45%;      /* Приглушенный текст */
```

### Темная тема (`.dark`)
```css
--background: 222 47% 5%;          /* Темный фон */
--foreground: 210 40% 98%;         /* Светлый текст */
--card: 222 47% 8%;                /* Темные карточки */
--border: 217 33% 17%;             /* Темные границы */
--muted: 217 33% 17%;              /* Приглушенный темный фон */
--muted-foreground: 215 20% 65%;   /* Приглушенный светлый текст */
```

## 🎯 Примеры использования CSS переменных

### Вместо хардкода:
```tsx
// ❌ Плохо - хардкодные цвета
<div className="bg-white text-gray-900 border-gray-200">
  <p className="text-gray-600">Текст</p>
</div>

// ✅ Хорошо - CSS переменные
<div className="bg-card text-foreground border-border">
  <p className="text-muted-foreground">Текст</p>
</div>
```

### Для уведомлений:
```tsx
// ❌ Плохо
<div className="bg-green-50 text-green-800 border-green-200">
  Успех!
</div>

// ✅ Хорошо - поддержка темной темы
<div className="bg-green-500/10 dark:bg-green-500/20 text-green-700 dark:text-green-300 border-green-500/30 dark:border-green-500/50">
  Успех!
</div>
```

## 🔍 Тестирование

### Как протестировать тему:

1. **Откройте сайт** в preview
2. **Найдите переключатель** в правом верхнем углу
3. **Нажмите на иконку луны** ☾
4. Сайт переключится на темную тему
5. **Обновите страницу** (F5)
6. Тема должна сохраниться (без мерцания)
7. **Переключитесь обратно** нажав на иконку солнца ☀️

### Проверка на разных страницах:
- Перейдите на `/catalog` - карточки авто
- Перейдите на `/reviews` - отзывы с бейджами
- Перейдите на `/faq` - аккордеон
- Перейдите на `/contacts` - формы
- Откройте админ-панель `/admin`

Все элементы должны корректно отображаться в обеих темах!

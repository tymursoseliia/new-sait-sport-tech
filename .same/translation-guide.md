## 🔄 Как переключить язык

### В интерфейсе сайта:

1. **Найдите переключатель языков** в правом верхнем углу хедера
2. **Нажмите на иконку глобуса** 🌐
3. **Выберите язык:**
   - 🇷🇺 RU - Русский
   - 🇬🇧 EN - English
   - 🇧🇾 BE - Беларуская

### Что происходит при переключении:

1. Весь контент на странице переводится на выбранный язык
2. Выбор сохраняется в `localStorage`
3. При следующем посещении сайт откроется на выбранном языке

## 🛠️ Как использовать переводы в коде

### 1. Импортировать хук:

```tsx
import { useLanguage } from '@/contexts/LanguageContext';
```

### 2. Использовать в компоненте:

```tsx
export default function MyComponent() {
  const { t } = useLanguage();

  return (
    <div>
      <h1>{t('hero.title')}</h1>
      <p>{t('hero.subtitle')}</p>
    </div>
  );
}
```

### 3. Для динамических значений:

```tsx
// Простой перевод
<span>{t('catalog.from')} {price.toLocaleString('ru-RU')} ₽</span>

// Условный перевод
<Badge>
  {status === 'available' ? t('catalog.available') : t('catalog.order')}
</Badge>
```

## 📝 Как добавить новый перевод

### Шаг 1: Добавить ключ во все языки

**ru.json:**
```json
{
  "mySection.newKey": "Текст на русском"
}
```

**en.json:**
```json
{
  "mySection.newKey": "Text in English"
}
```

**be.json:**
```json
{
  "mySection.newKey": "Тэкст на беларускай"
}
```

### Шаг 2: Использовать в компоненте

```tsx
<p>{t('mySection.newKey')}</p>
```

## 📚 Полезные команды

### Проверить переводы:
```bash
# Найти все использования t()
grep -r "t('" src/

# Найти непереведенные строки
grep -r "\"[А-Яа-я]" src/ --include="*.tsx"
```

### Добавить новый язык:

1. Создать файл `/src/locales/новый-язык.json`
2. Скопировать структуру из `ru.json`
3. Перевести все ключи
4. Добавить в `LanguageContext.tsx`:
```tsx
type Language = 'ru' | 'en' | 'be' | 'новый-код';
```
5. Обновить переключатель языков в Header

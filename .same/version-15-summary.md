### 1. Добавлены заголовки для отключения кэширования
**Файлы:**
- `src/app/api/admin/cars/route.ts`
- `src/app/api/admin/reviews/route.ts`
- `src/app/api/admin/articles/route.ts`

**Изменения:**
```typescript
return NextResponse.json(
  { success: true, data: items },
  {
    headers: {
      'Cache-Control': 'no-store, no-cache, must-revalidate, max-age=0',
      'Pragma': 'no-cache',
      'Expires': '0',
    },
  }
);
```

**Почему это важно:**
Браузеры и Next.js кэшируют API ответы для оптимизации. Эти заголовки заставляют всегда загружать свежие данные из базы.

## 🐛 Решенная проблема:

**До:**
1. Пользователь удаляет автомобиль в админ-панели
2. Автомобиль исчезает из списка
3. Пользователь обновляет страницу (F5)
4. ❌ Автомобиль появляется снова

**После:**
1. Пользователь удаляет автомобиль в админ-панели
2. Автомобиль исчезает из списка
3. Появляется алерт "Автомобиль успешно удалён!"
4. Пользователь обновляет страницу (F5)
5. ✅ Автомобиль НЕ появляется (остается удаленным)

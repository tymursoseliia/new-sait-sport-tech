# Настройки яркости фонового видео - Финальная версия

## ✅ Что сделано (Версия 14)

### 1. Увеличена яркость видео на странице "О компании"

**Файл:** `src/components/VideoBackground.tsx`

**Текущие настройки:**
```tsx
// Прозрачность видео
className="w-full h-full object-cover opacity-70"
// Было: opacity-30 (30%)
// Стало: opacity-70 (70%)
```

**Overlay градиенты:**
```tsx
const overlayClasses = {
  light: 'bg-gradient-to-r from-zinc-900/40 to-zinc-900/20',   // 40% → 20%
  medium: 'bg-gradient-to-r from-zinc-900/80 to-zinc-900/60',  // 80% → 60%
  dark: 'bg-gradient-to-r from-zinc-900/90 to-zinc-900/70',    // 90% → 70%
};
```

### 2. Применение на странице /about

**Файл:** `src/app/about/page.tsx`

```tsx
<VideoBackground
  videoSrc="/videos/about-bg.mp4"
  fallbackImage="https://images.unsplash.com/photo-1492144534655-ae79c964c9d7?w=1920"
  overlayOpacity="light"  // ← Используется LIGHT режим
  useVideo={true}         // ← Видео ВКЛЮЧЕНО
/>
```

## 📊 Сравнение настроек

| Параметр | Было | Стало | Изменение |
|----------|------|-------|-----------|
| Прозрачность видео | 30% | **70%** | +40% |
| Overlay начало | 80% | **40%** | -40% |
| Overlay конец | 60% | **20%** | -40% |
| Режим overlay | medium | **light** | Светлее |

## 🎨 Визуальный эффект

**До изменений:**
- Видео едва заметно (30% прозрачности)
- Сильное затемнение overlay (80-60%)
- Общий эффект: очень темный фон

**После изменений:**
- Видео хорошо видно (70% прозрачности)
- Легкое затемнение overlay (40-20%)
- Общий эффект: светлый, динамичный фон с хорошей читаемостью текста

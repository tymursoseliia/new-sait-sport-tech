## 📋 Как использовать

### Для включения видео на главной странице:

1. Поместите видео в `public/videos/hero-video.mp4`
2. В файле `src/app/page.tsx` измените:
   ```tsx
   const USE_VIDEO = true; // строка ~57
   ```

### Для включения видео на странице "О компании":

1. Поместите видео в `public/videos/about-bg.mp4`
2. В файле `src/app/about/page.tsx` измените:
   ```tsx
   const USE_VIDEO = true; // строка ~179
   ```

## 🎬 Требования к видео

**Формат:** MP4 (H.264)
**Разрешение:** 1920x1080 (Full HD)
**Длительность:** 10-30 секунд
**Размер:** до 10 МБ
**Содержание для главной:** автомобили, дороги Европы
**Содержание для about:** команда, офис, процесс работы

## 🛠 Технические детали

### Структура компонента VideoBackground

```tsx
<VideoBackground
  videoSrc="/videos/about-bg.mp4"          // путь к видео
  fallbackImage="https://..."               // fallback изображение
  overlayOpacity="medium"                   // light/medium/dark
  useVideo={USE_VIDEO}                      // включить/выключить
/>
```

### Стили секции с видео

```tsx
<section className="relative ... overflow-hidden">
  <VideoBackground ... />
  <div className="relative z-10">
    {/* Контент поверх видео */}
  </div>
</section>
```

## 🔄 Переиспользование

Компонент VideoBackground можно использовать на любой странице:

```tsx
import VideoBackground from '@/components/VideoBackground';

<section className="relative overflow-hidden">
  <VideoBackground
    videoSrc="/videos/my-video.mp4"
    fallbackImage="https://example.com/image.jpg"
    overlayOpacity="medium"
    useVideo={true}
  />
  <div className="relative z-10">{/* контент */}</div>
</section>
```

## 📝 Примечания

- По умолчанию видео **отключено** (`USE_VIDEO = false`)
- При отсутствии видео автоматически показывается fallback изображение
- Видео без звука - обязательное требование для autoplay
- Компонент обрабатывает ошибки загрузки видео
- Главная страница остается без изменений (кроме рефакторинга)

# 🎥 Инструкция по загрузке видео на главную страницу

## ✅ Готово

Код обновлен! Теперь на главной странице вместо фонового изображения будет проигрываться ваше видео.

---

## 📁 Как загрузить видео

### Вариант 1: Через интерфейс Same (Рекомендуется)

1. **Найдите ваш видеофайл** на компьютере
2. **Перетащите его** в левую панель Same (File Explorer)
3. **Поместите в папку** `public/videos/`
4. **Переименуйте файл** в `hero-video.mp4`

### Вариант 2: Через терминал

```bash
# Загрузите видео в папку public/videos/
cp /путь/к/вашему/видео.mp4 volga-auto-premier/public/videos/hero-video.mp4
```

---

## 🎬 Требования к видео

### Рекомендуемые параметры:

**Формат:**
- ✅ MP4 (H.264 кодек) - лучшая совместимость
- ✅ WebM - альтернатива для современных браузеров

**Разрешение:**
- ✅ **1920x1080** (Full HD) - оптимально
- ✅ **1280x720** (HD) - если нужен меньший размер файла
- ⚠️ Не используйте 4K - слишком большой размер

**Длительность:**
- ✅ **10-30 секунд** - оптимально для зацикленного фона
- ⚠️ Видео будет зацикливаться автоматически

**Размер файла:**
- ✅ **Максимум 10-15 МБ** - для быстрой загрузки
- ⚠️ Если файл больше - сожмите его (см. раздел "Сжатие видео")

**Ориентация:**
- ✅ **Горизонтальная** (16:9)
- ✅ Видео будет обрезано под размер экрана (object-fit: cover)

---

## 🔧 Сжатие видео (если файл слишком большой)

### Онлайн инструменты:
- [FreeConvert](https://www.freeconvert.com/video-compressor) - бесплатно
- [Clipchamp](https://clipchamp.com/en/video-compressor/) - бесплатно
- [HandBrake](https://handbrake.fr/) - программа для сжатия (бесплатно)

### Через FFmpeg (для опытных):
```bash
ffmpeg -i input.mp4 -vcodec h264 -acodec aac -b:v 2M -b:a 128k hero-video.mp4
```

**Параметры сжатия:**
- Битрейт видео: 2-4 Mbps
- Разрешение: 1920x1080 или 1280x720
- FPS: 24-30
- Убрать звук (он всё равно отключен на сайте)

## 🎨 Как выглядит видео на сайте

### Характеристики отображения:
- ✅ **Автоматическое воспроизведение** - начинается сразу
- ✅ **Зацикливание** - бесконечно повторяется
- ✅ **Без звука** - muted (требование браузеров для autoplay)
- ✅ **Затемнение 70%** - чтобы текст был читаемым
- ✅ **Адаптивное** - подстраивается под любой размер экрана

### Градиентный overlay:
Поверх видео накладывается темный градиент, чтобы:
- Текст был хорошо читаемым
- Видео не отвлекало от контента
- Сохранялась атмосфера премиальности

## 📝 Какое видео использовать

### Рекомендации по контенту:

**✅ Хорошие варианты:**
- Красивая езда по европейским дорогам
- Панорамная съемка автомобилей

**❌ Избегайте:**

## 🔄 Поддержка нескольких форматов

Если хотите максимальную совместимость, можете добавить WebM версию:

```html
<video autoPlay loop muted playsInline>
  <source src="/videos/hero-video.webm" type="video/webm" />
  <source src="/videos/hero-video.mp4" type="video/mp4" />
</video>
```

Загрузите оба файла:
- `public/videos/hero-video.mp4`
- `public/videos/hero-video.webm`

## 🚨 Если видео не работает

### Проверьте:
- ✅ Файл называется **точно** `hero-video.mp4`
- ✅ Файл находится в папке `public/videos/`
- ✅ Формат файла - MP4 (H.264)
- ✅ Размер файла не больше 50 МБ
- ✅ Перезагрузили страницу (Ctrl+F5)

### Консоль браузера:
Откройте консоль (F12) и проверьте ошибки:
- Если видит 404 - файл не найден, проверьте путь
- Если проблема с кодеком - пересохраните в MP4 H.264

### Шаг 3: Подтверждение прав на сайт

Яндекс предложит несколько способов. Рекомендую **Meta-тег** (самый простой):

#### 🎯 Способ 1: Meta-тег (Рекомендуется)

1. **Скопируйте код подтверждения:**
   ```html
   <meta name="yandex-verification" content="ваш_уникальный_код" />
   ```

2. **Отправьте мне этот код** - я добавлю его в `<head>` сайта

3. **После добавления кода:**
   - Я закоммичу изменения
   - Подожду деплой на Vercel (2-3 минуты)
   - Вы нажмёте "Проверить" в Яндекс Вебмастере

#### 📄 Способ 2: HTML-файл

1. **Скачайте файл** от Яндекса (например, `yandex_12345.html`)
2. **Загрузите его в корень сайта** (в папку `public/`)
3. **Проверьте доступность:**
   ```
   https://www.volga-auto-premier.com/yandex_12345.html
   ```
4. **Нажмите "Проверить"** в Вебмастере

#### 🌐 Способ 3: DNS-запись (для продвинутых)

1. **Скопируйте TXT-запись** от Яндекса
2. **Добавьте её в DNS вашего домена** (через регистратора)
3. **Подождите 24 часа** (распространение DNS)
4. **Нажмите "Проверить"**

### Шаг 4: Главное зеркало сайта

После подтверждения прав:

1. **Перейдите в раздел "Индексирование → Переезд сайта"**

2. **Укажите главное зеркало:**
   ```
   https://www.volga-auto-premier.com
   ```

   ✅ С `https://` и `www.`

3. **Настройте 301 редирект** (если ещё не сделано):
   - С `http://volga-auto-premier.com` → `https://www.volga-auto-premier.com`
   - С `https://volga-auto-premier.com` → `https://www.volga-auto-premier.com`

   **Это обычно делается на уровне Vercel** - проверю настройки.

2. **Нажмите "Добавить sitemap"**

3. **Введите URL:**

## 🎯 Важные настройки после подтверждения

### 1. Турбо-страницы (по желанию)

Яндекс Турбо - это мобильные AMP-версии страниц для быстрой загрузки.

**Где настроить:**
- Вебмастер → Турбо-страницы

**Нужно ли:**
- ⚠️ Необязательно для автопригона
- ✅ Сайт уже быстрый и адаптивный

### 2. Микроразметка

**Проверьте наличие:**
- Вебмастер → Инструменты → Валидатор микроразметки

**Введите URL:**
```
https://www.volga-auto-premier.com
```

**Что должно быть:**
- ✅ Organization (организация)
- ✅ LocalBusiness (местная компания)
- ✅ BreadcrumbList (хлебные крошки)
- ✅ Product (автомобили в каталоге)
- ✅ Review (отзывы клиентов)

### 3. Скорость загрузки

**Проверьте производительность:**
- Вебмастер → Диагностика → Скорость загрузки

**Оптимальные показатели:**
- ✅ Скорость загрузки: менее 3 сек
- ✅ First Contentful Paint: менее 1.8 сек
- ✅ Time to Interactive: менее 3.8 сек

### 4. Безопасность HTTPS

**Проверьте сертификат:**
- Вебмастер → Безопасность → HTTPS

**Должно быть:**
- ✅ Действующий SSL-сертификат
- ✅ Все ресурсы загружаются через HTTPS
- ✅ Нет смешанного контента (mixed content)

## 📊 Мониторинг и отчёты

### Еженедельно проверяйте:

1. **Индексирование:**
   - Вебмастер → Индексирование → Страницы в поиске
   - Смотрите динамику роста

2. **Поисковые запросы:**
   - Вебмастер → Поиск → Поисковые запросы
   - Анализируйте по каким словам находят сайт

3. **Внешние ссылки:**
   - Вебмастер → Ссылки → Внешние ссылки
   - Контролируйте качество обратных ссылок

4. **Ошибки:**
   - Вебмастер → Диагностика → Ошибки обхода
   - Исправляйте 404 и другие ошибки

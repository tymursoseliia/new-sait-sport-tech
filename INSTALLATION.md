# 📦 Инструкция по установке ВОЛГА-АВТО

## Шаг 1: Распаковать архив

```bash
tar -xzf volga-auto-premier-source.tar.gz
cd volga-auto-premier
```

## Шаг 2: Установить Bun (если ещё не установлен)

### macOS / Linux:
```bash
curl -fsSL https://bun.sh/install | bash
```

### Windows (WSL):
```bash
powershell -c "irm bun.sh/install.ps1 | iex"
```

Проверьте установку:
```bash
bun --version
```

## Шаг 3: Установить зависимости

```bash
bun install
```

Это займёт 1-2 минуты и загрузит все необходимые пакеты (~718 МБ).

## Шаг 4: Запустить проект

### Режим разработки:
```bash
bun run dev
```

Проект будет доступен по адресу: **http://localhost:3000**

### Режим production:
```bash
bun run build
bun run start
```

## 📋 Полезные команды

- `bun run dev` - запуск в режиме разработки (Turbopack)
- `bun run build` - сборка для production
- `bun run start` - запуск production сервера
- `bun run lint` - проверка кода на ошибки

## 🔧 Требования

- **Node.js**: 18.17 или выше (или Bun)
- **Bun**: 1.0.0 или выше (рекомендуется)
- **ОЗУ**: минимум 2 ГБ свободной памяти

## 🌍 Настройка языков

Проект поддерживает русский и английский языки.
Переключение доступно в шапке сайта (флаг RU/EN).

## 🎨 Настройка темы

Переключение между светлой и тёмной темой - кнопка в шапке (иконка солнца/луны).

Для изменения основного акцентного цвета всего сайта:
- Откройте `src/app/globals.css`
- Найдите переменную `--brand-accent: 217 91% 35%;`
- Измените значение HSL на нужный цвет

## ❓ Проблемы и решения

### Проблема: "Cannot find module"
**Решение:** Удалите node_modules и переустановите:
```bash
rm -rf node_modules
bun install
```

### Проблема: Порт 3000 занят
**Решение:** Используйте другой порт:
```bash
bun run dev -- -p 3001
```

### Проблема: Ошибки TypeScript
**Решение:** Проверьте код линтером:
```bash
bun run lint
```

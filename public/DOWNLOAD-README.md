## 🚀 Инструкция по установке:

### Шаг 1: Распаковка

**Windows:**
```bash
# Распакуйте архив через проводник или:
tar -xzf volga-auto-premier-optimized.zip
```

**Linux / macOS:**
```bash
tar -xzf volga-auto-premier-optimized.tar.gz
```

### Шаг 2: Установка зависимостей

Перейдите в папку проекта:
```bash
cd volga-auto-premier
```

Установите Bun (если еще не установлен):
```bash
curl -fsSL https://bun.sh/install | bash
```

Установите зависимости:
```bash
bun install
```

### Шаг 3: Настройка переменных окружения

Создайте файл `.env.local` на основе примера:
```bash
cp .env.local.example .env.local
```

Отредактируйте `.env.local` и добавьте ваши данные:
- `TELEGRAM_BOT_TOKEN` - токен вашего Telegram бота
- `TELEGRAM_CHAT_ID` - ID чата для уведомлений

### Шаг 4: Запуск проекта

**Режим разработки:**
```bash
bun run dev
```

Откройте браузер: [http://localhost:3000](http://localhost:3000)

**Production сборка:**
```bash
bun run build
bun start
```

# 📁 Папка для видеофайлов сайта

## Необходимые видеофайлы

### 1. hero-video.mp4
**Для главной страницы** - фоновое видео в hero секции

**Требования:**
- Формат: MP4 (H.264)
- Разрешение: 1920x1080 (Full HD) или 1280x720
- Длительность: 10-30 секунд (проигрывается в цикле)
- Размер: до 10-15 МБ
- Содержание: автомобили премиум-класса, дороги Европы

**Как включить:**
1. Поместите видео в `public/videos/hero-video.mp4`
2. В файле `src/app/page.tsx` измените `USE_VIDEO = false` на `USE_VIDEO = true`

---

### 2. about-bg.mp4
**Для страницы "О компании"** - фоновое видео в секции истории компании

**Требования:**
- Формат: MP4 (H.264)
- Разрешение: 1920x1080 (Full HD) или 1280x720
- Длительность: 10-30 секунд (проигрывается в цикле)
- Размер: до 10-15 МБ
- Содержание: команда за работой, офис, процесс подбора автомобилей

**Как включить:**
1. Поместите видео в `public/videos/about-bg.mp4`
2. В файле `src/app/about/page.tsx` измените `USE_VIDEO = false` на `USE_VIDEO = true`

---

## Оптимизация видео

Для сжатия видео используйте FFmpeg:

```bash
# Базовое сжатие
ffmpeg -i input.mp4 -c:v libx264 -crf 28 -preset slow -vf scale=1920:-2 -an output.mp4
```
